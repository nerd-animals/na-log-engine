📽️ Project

숫자 야구(bulls and cows) 개발

python

숫자 야구는 3자리의 숫자를 맞추는 간단한 게임입니다. 평소 즐겨하는 게임으로 개발에 재미를 붙이고자 간단하게 만들어보았습니다. curses라는 터미널 그래픽을 만들 수 있는 모듈로 구현하였고, 게임을 좀 더 간단하게 실행시키고자 pyinstaller 라는 python 패키지를 이용하였습니다.

  • raccoon : 프로젝트 계획 및 관리, 코드의 방향성 제시

  • otter : 개발 담당
