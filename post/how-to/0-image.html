이번 포스팅에서는 NA-Log에서 Image 사용 방법에 대해 알려드리겠습니다.

전반적인 folder 구조는 다음과 같습니다.

게시글에서 public에 있는 이미지를 사용하고 싶다면, public/을 기본 경로라고 생각하시면 됩니다.
예를 들어, 아래와 같이 public/crazy-gugu.jpg가 있다고 가정하겠습니다.

1. Markdown 문법 사용하기

NA-Log는 Github-Flavored-Markdown 문법을 지원합니다.
따라서, 아래와 같이 사용할 수 있습니다.

Web에 있는 이미지 사용

2. HTML 태그 사용하기 (비권장 😥)

GFM에서 HTML 태그를 사용할 수 있으므로 img태그를 사용한 이미지 첨부가 가능합니다.
이미지 경로에 basePath를 적용해주어야 하는데, HTML 태그로 입력할 경우 처리가 안 됩니다.

그러나 세부 옵션을 조정하려면, HTML 태그를 사용해야 합니다.
HTML 태그를 사용할 경우, basePath를 직접 추가해주세요!
basePath는 본인의 블로그 링크에서 확인해주시면 됩니다.
<Github Owner>.github.io/<basePath>

NA-Log에서 이미지를 첨부하는 방법에 대해 소개해드렸습니다.
문의 사항이 있으시다면, 댓글 남겨주세요! 👍
